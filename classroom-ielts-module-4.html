Module 4: Writing (Task 1 & Task 2)

Master task response, organization, and language range.

✍ Writing⏱ 90–120 min📝 15 Quiz Questions🎯 Module 4/6

Module 4 of 6

Module Objectives

  • Differentiate Task 1 vs Task 2 requirements.
  • Use strong paragraph structure and cohesion.
  • Apply band descriptors for self-checking.

Module 4 Quick Start

  • Task split: Plan 20 minutes for Task 1 and 40 minutes for Task 2.
  • Score priorities: Focus on task response, coherence, lexical range, and grammar accuracy.
  • Write then improve: Build a draft, then edit for clarity and precision.
  • Use evidence: In Task 1 report key trends only; in Task 2 support claims with examples.

Writing Task Guide

Task 1

Write an introduction, overview, and two detail paragraphs with comparisons. Summarize the main trend from the visual data — do not add opinions.

  • Pattern 1: Paraphrase + Overview — "The chart demonstrates… The data compares…"
  • Pattern 2: Paraphrase + Context + Overview — "The diagram illustrates… Between X and Y…"

Task 2

State a clear position, develop two focused body paragraphs, and conclude logically. Each body paragraph: topic sentence + explanation + example + link back.
