What Is a Spreadsheet?

Microsoft Excel is a spreadsheet program used to organize, calculate, and analyze data. A spreadsheet is made up of cells arranged in rows (numbered 1, 2, 3…) and columns (lettered A, B, C…). Each cell has an address like A1, B3, or C12.

Excel is used for: personal budgets, business finances, tracking sales, managing inventories, student grade books, project timelines, and much more.

Entering and Formatting Data

Click any cell and start typing to enter data. Press Enter to move to the next row, or Tab to move to the next column.

Data types:

  • Text: Names, labels, headings — left-aligned by default.
  • Numbers: Quantities, prices, scores — right-aligned by default.
  • Dates: Excel recognizes formats like 25/06/2026 and can calculate with them.

Format cells using the Home tab: bold headers, change font size, add borders, or apply number formats (currency, percentage, date).

Basic Formulas and Functions

Formulas always start with an = (equals sign).

Essential formulas:

  • =A1+B1 — Adds two cells
  • =SUM(A1:A10) — Adds a range of cells
  • =AVERAGE(B1:B5) — Calculates the average
  • =MAX(C1:C20) — Finds the highest value
  • =MIN(C1:C20) — Finds the lowest value
  • =COUNT(D1:D50) — Counts how many cells have numbers

To apply a formula to many rows at once, type it in the first row, then drag the small square at the bottom-right corner of the cell downward.

Creating a Simple Budget

A basic personal budget in Excel has four columns: Category, Budgeted Amount, Actual Amount, Difference.

Steps:

  1. List your income sources and expense categories in column A.
  2. Enter your planned amounts in column B.
  3. Track actual spending in column C.
  4. In column D, use a formula like =B2-C2 to see if you are over or under budget.
  5. Use a SUM formula at the bottom of each column for totals.
Color-coding cells (red for over budget, green for under) makes the spreadsheet easier to read at a glance.

💡 Key Tips

  • Press F2 to edit a cell without retyping everything.
  • Use Ctrl+1 to open the Format Cells dialog for detailed formatting options.
  • Freeze the top row (View → Freeze Panes) so headers stay visible when you scroll down a long spreadsheet.
  • Use filters (Data → Filter) to quickly sort and search large datasets.
