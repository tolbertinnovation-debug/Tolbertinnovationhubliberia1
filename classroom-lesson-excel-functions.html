📊 Excel Functions & Lookup Formulas

Unlock Excel's power with IF statements, VLOOKUP, SUMIF, and data analysis tools that professionals use to process data quickly and accurately.

Learning Objectives

  • Understand: Logical Functions — IF, AND, OR
  • Understand: Lookup Functions — VLOOKUP and XLOOKUP
  • Understand: Conditional Aggregation — SUMIF, COUNTIF, AVERAGEIF
  • Understand: Data Validation and Pivot Tables

Logical Functions: IF, AND, OR

The IF function is the most important formula in Excel. It tests a condition and returns one value if true and another if false. Syntax: =IF(condition, value_if_true, value_if_false). Example: if a student's score is in column B and you want to show "Pass" or "Fail":

=IF(B2>=50, "Pass", "Fail")

You can nest IFs for multiple conditions: =IF(B2>=70, "Distinction", IF(B2>=50, "Pass", "Fail")). AND and OR allow multiple conditions: =IF(AND(B2>=50, C2="Attended"), "Pass", "Fail") — the student passes only if they scored 50+ AND attended. =IF(OR(B2>=90, C2="Scholarship"), "Award", "No Award") — they receive an award if either condition is true.

These functions are used in business for: flagging overdue payments, calculating bonuses, grading systems, inventory alerts, and loan eligibility checks. Mastering IF means you can automate decisions that would otherwise require manual checking of every row.

VLOOKUP and XLOOKUP

VLOOKUP (Vertical Lookup) searches for a value in the first column of a table and returns a value from another column in the same row. It is used to pull data from one table into another — like looking up an employee's salary by their ID number.

=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])

Example: You have a list of student IDs in column A and you want to pull their names from a separate table. =VLOOKUP(A2, StudentTable, 2, FALSE) — looks up the value in A2, searches in StudentTable, and returns the value from column 2 (name). Always use FALSE for exact match. VLOOKUP only searches left-to-right; the lookup column must be the leftmost column of your table.

XLOOKUP (available in Excel 365 and newer) is more powerful — it can search in any direction and is simpler to write. =XLOOKUP(A2, StudentTable[ID], StudentTable[Name], "Not Found"). If your Excel version supports XLOOKUP, prefer it over VLOOKUP. HLOOKUP works horizontally (across rows) instead of vertically. INDEX/MATCH is a more flexible alternative for complex lookups.

SUMIF, COUNTIF, and AVERAGEIF

These "conditional aggregation" functions calculate totals, counts, or averages only for rows that meet a specific condition — extremely useful for business reports and financial analysis.

=SUMIF(range, criteria, sum_range)

Example: You have sales data with a "Region" column and a "Sales" column. To total sales for only the Monrovia region: =SUMIF(B:B, "Monrovia", C:C). COUNTIF counts how many rows match: =COUNTIF(B:B, "Monrovia") — how many transactions were in Monrovia. AVERAGEIF averages only matching rows: =AVERAGEIF(B:B, "Monrovia", C:C) — average sales per Monrovia transaction.

SUMIFS, COUNTIFS, and AVERAGEIFS (with an S) allow multiple conditions. =SUMIFS(C:C, B:B, "Monrovia", D:D, "Q1") — total Monrovia sales in Q1 only. These formulas replace hours of manual filtering and are standard in every financial model and business dashboard.

Data Validation and Pivot Tables

Data Validation restricts what can be entered in a cell, preventing errors at the source. Select a cell > Data > Data Validation. You can restrict entries to: a list (dropdown menu of allowed values), whole numbers within a range, dates within a period, or custom formulas. Add an error message to tell users what is acceptable. Dropdowns in forms ensure consistent data entry — for example, ensuring a "Status" column only accepts "Open", "Closed", or "Pending".

Pivot Tables are Excel's most powerful data analysis tool. They summarize large datasets interactively — grouping, totaling, and filtering thousands of rows in seconds. To create: select your data > Insert > PivotTable. In the field list, drag fields to Rows, Columns, Values, and Filters. Example: drag "Region" to Rows, "Product" to Columns, and "Sales" to Values — you get a cross-tab showing sales by region and product with grand totals automatically.

PivotCharts turn a Pivot Table into an interactive chart. Slicers (Insert > Slicer) add visual filter buttons to your Pivot Table. These tools allow you to explore data without writing any formulas — they are used in every professional business reporting context.

💡 Key Tips

  • Use absolute references ($A$1) in VLOOKUP table arrays so the formula does not break when copied down — press F4 to toggle between reference types.
  • Named Ranges (Formulas > Define Name) make formulas readable — =SUMIF(Region, "Monrovia", Sales) is clearer than =SUMIF(B:B, "Monrovia", C:C).
  • Press Ctrl+T to convert a data range into an Excel Table — it auto-expands formulas and makes Pivot Tables much easier to refresh.
  • When a formula returns #N/A or #VALUE!, wrap it in IFERROR: =IFERROR(VLOOKUP(...),"Not Found") to show a friendly message instead of an error.
  • Conditional Formatting (Home > Conditional Formatting) highlights cells that meet criteria with color — great for flagging late payments or low stock.
