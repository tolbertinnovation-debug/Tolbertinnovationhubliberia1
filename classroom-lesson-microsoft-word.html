Getting Started with Microsoft Word

Microsoft Word is the world's most widely used word-processing software. It is used to create documents including letters, reports, CVs, proposals, and more.

When you open Word, you see the Ribbon, the toolbar at the top with tabs: Home, Insert, Design, Layout, References, etc. Each tab has related tools grouped together. The large white area in the middle is your document canvas.

Formatting Text

Formatting makes your document professional and easy to read.

Basic text formatting (Home tab):

  • Bold (Ctrl+B): Makes text stand out for headings or key words.
  • Italic (Ctrl+I): Used for titles of books, emphasis, or foreign words.
  • Underline (Ctrl+U): Used sparingly for links or special emphasis.
  • Font & Size: Use a professional font like Calibri, Arial, or Times New Roman at 11–12pt for body text.
  • Alignment: Left-align body text; center headings where appropriate.

Paragraphs, Lists, and Headings

Paragraphs: Press Enter once to start a new paragraph. Use consistent spacing; avoid pressing Enter multiple times to create space; instead adjust paragraph spacing in the Layout tab.

Lists: Use bulleted lists (•) for unordered items, numbered lists for steps or sequences. Find these in the Home tab.

Headings: Apply Heading 1 and Heading 2 styles (in the Styles panel on the Home tab) to structure your document. This also helps you navigate long documents using the Navigation Pane (View → Navigation Pane).

Saving, Printing, and Sharing

Save your work: Press Ctrl+S frequently. The first time you save, choose a filename and location. Word saves files as .docx by default.

Save as PDF: File → Save As → choose PDF. Send PDFs for formal submissions, as they look the same on every device and cannot easily be edited.

Print: File → Print (or Ctrl+P). Check the print preview before printing: verify margins, page numbers, and that content fits properly.

Share: Use File → Share to send via email directly, or save to OneDrive for cloud sharing.

💡 Key Tips

  • Press Ctrl+Z to undo any mistake immediately.
  • Use Ctrl+A to select all text, then apply formatting to the whole document at once.
  • Use the Format Painter (paintbrush icon) to copy formatting from one section and apply it to another.
  • Turn on "Show/Hide" (¶ button) to see hidden characters like spaces and paragraph marks, useful for fixing formatting issues.

📝 Practice Assignment

Open Microsoft Word (or Google Docs if you don't have Word). Create a one-page document about yourself with: (1) A centered heading with your full name in a large bold font. (2) A short "About Me" paragraph (3–5 sentences). (3) A bulleted list of your top 5 skills. (4) A numbered list of your 3 career goals. Save it as both a .docx and a PDF.
